## 🏃 Quick Start

**Terminal 1 (Backend):**
```bash
nvm use 20
cd /Volumes/AHARDRIVE/Apps/Projects/TapTask/server
npx tsx watch src/_core/index.ts
```

**Terminal 2 (Frontend):**
```bash
nvm use 20
cd /Volumes/AHARDRIVE/Apps/Projects/TapTask/client
npx vite
```
